# 🎮 Tic Tac Toe

A modern, responsive Tic Tac Toe game built with pure HTML, CSS, and JavaScript.

## 🚀 Features
- Interactive 3x3 grid
- Player vs Player gameplay
- Win & draw detection
- Restart functionality
- Clean UI with hover effects

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
